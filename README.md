# obs

> An setup for use with the OBS "browser" setting.

## features

- designed around 1080p
- two green screen areas sized for 1280x1024 screenshare and 4:3 webcam
- timer pomodoro section controllable by URL params (`?mode=work` | `?mode=break`)
- if no mode is supplied, no timer displays
- timer emoji :timer_clock: rain + alarm sound on alarm end
- cancel timer and trigger end sequence with `ctrl` + `c`
